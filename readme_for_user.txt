运行前提：
1. C盘根目录新建文件夹ause。(程序文件路径为C:\\ause)
2. 声音文件和.exe文件在同一文件夹下（此下载包默认）

操作提示：
键盘交互获取键盘字符，需要输入法处于英文输入法状态，中文输入法可能操作错误。
在所有菜单界面按W，S可以上下移动光标，回车键确认。在主菜单可按数字键快速跳转光标。
游戏模块可以从查询学号（1），查询姓名（2）处进入，也可以从密码系统菜单（8）里进入。注意进入提示“按F进入”

密码系统：
默认密码（重置密码）：教师端000000，学生端111111
管理员密码：580580
密码输入界面回退键可删除，输入后回车键确认，模拟正常输入界面。

注意事项：
程序采用了一些延时函数Sleep，如果发现键入还没反应，可能是在执行Sleep函数，
Sleep函数延时时间不超过2.5秒，可以稍稍停顿。
文件采用了字符文件，便于信息查阅，但不建议直接修改C:\\ause中的文件，可能会造成格式错误。
修改可从程序的修改信息和修改密码操作。如果不慎直接打开.txt错误编辑，可直接删除文件，程序会检测无文件然后重新自动生成文件。

该程序移植到其他电脑测试通过，可以正常运行。
